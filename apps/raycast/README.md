# Port Manager Raycast Extension

This is the local Raycast client for Port Manager. It imports `@patrickjs/port-manager`, so Raycast commands and AI tools share the same scanner, common-port labels, kill implementation, and cooperative reservation registry as the CLI and macOS app.

```sh
npm install
npm run dev
```

Use `npm run build` after development if you want Raycast to keep the local extension without the dev server.
